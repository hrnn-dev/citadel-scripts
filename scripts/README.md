#Citadel
Infraestructure as code (IaC) para el despliegue y optimización de nodos handless Debian.

##Descripción
Este repositorio contiene la configuración, scripts de automatización y herramientas de orquestación para el 
despliegue de 'Citadel', un nodo basado en Debian GNU/Linux 12.

##Componentes de infraestructura.
-**Sistema Operativo:** Debian 12 (Hardened Headless)
-**Gestión: ** SSH, Tmux Multiplexing
-**Herramientas de red: ** Nmap, Tcpdump, Scapy, etc.
-**Entorno: ** Bash scripts para despliegue automatizado, Python, C, Rust.

##Versionado
Este laboratorio sigue una filosofía de versionado incremental. Cada script modificado es registrado para
asegurar la reproducibilidad del entorno ante contingencias.
